# Amazon ReLife

Amazon ReLife is a circular-commerce platform powered by Product Digital Twins.

## Applications

- `frontend/customer-app` - customer experience and Product Twin
- `frontend/ops-dashboard` - operations dashboard
- `frontend/seller-dashboard` - seller dashboard
- `frontend/executive-dashboard` - executive dashboard

## Development

```bash
npm install
npm run dev:customer
```

Open `http://localhost:3000/product-twin/P123`.
